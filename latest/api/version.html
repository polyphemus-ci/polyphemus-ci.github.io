Version Information¶

Version information about polyphemus and its dependencies.

class polyphemus.version.version_info[source]¶

A representation of version information.

polyphemus.version.report_versions()[source]¶

Creates a string that reports the version of polyphemus and all its dependencies.

polyphemus.version.version_parser(ver)[source]¶

Parses a nominal version string into a version_info object. e.g. ‘0.20dev’ -> version_info(0, 20, 0, ‘dev’).
